# Data-Analysis-Project
Retail Customer Behavior &amp; Shopping Trends Analysis.

This project focuses on analyzing retail customer purchasing behavior to identify shopping patterns, customer preferences, and key trends. Excel is used for initial data organization and preprocessing, SQL is applied for data querying and extraction, and Python is used for data cleaning and exploratory data analysis. Power BI is utilized to create interactive dashboards and visualizations that present insights clearly and support data-driven decision-making in the retail domain.
